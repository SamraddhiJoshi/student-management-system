# 🎓 Student Management System

A web-based Student Management System built using PHP and MySQL that allows users to perform CRUD (Create, Read, Update, Delete) operations on student records.

## ✨ Features

- Add new student records
- View all student records
- Update existing student information
- Delete student records
- Secure database connectivity
- User-friendly interface

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- XAMPP

## 📂 Project Structure

```text
student-management-system/
│
├── db.php
├── index.php
├── form.php
├── delete.php
└── README.md
```
## ⚙️ Installation & Setup

1. Install XAMPP.
2. Start Apache and MySQL.
3. Create a MySQL database.
4. Import the students table.
5. Update database credentials in `db.php`.
6. Place the project inside the `htdocs` folder.
7. Open:

```text
http://localhost/student_system
```
